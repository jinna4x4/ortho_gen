# Ortho Gen - 牙科 3D 模型切削工具

這是一個簡單的工具，用於讀取 STL 模型並進行布林運算（Boolean Subtraction）。

## Mac 使用者安裝與執行教學 (新手簡易版)

如果您是使用 Mac 電腦且不熟悉程式操作，請依照下列步驟一步一步進行。

### 第一步：下載程式碼

1. 在此 GitHub 頁面右上方點擊綠色的 **< > Code** 按鈕。
2. 選擇 **Download ZIP**。
3. 下載完成後，到您的「下載項目 (Downloads)」資料夾，點兩下 `ortho_gen-main.zip` 解壓縮。
4. 您會看到一個資料夾叫做 `ortho_gen-main` (或是 `ortho_gen`)。

### 第二步：開啟終端機 (Terminal)

1. 按住鍵盤上的 `Command (⌘)` 鍵不放，然後按一下 `空白鍵 (Space)`，會跳出 Spotlight 搜尋框。
2. 輸入 `Terminal`，然後按 `Enter` 鍵。這會開啟一個白底或黑底的文字視窗（終端機）。

### 第三步：安裝設定 (只需做一次)

請依照順序執行以下指令。

1. **進入資料夾**：
   - 在終端機打字輸入 `cd ` (**注意：cd 後面要有一個空白鍵**)。
   - **不要**按 Enter，先打開 Finder (檔案總管) 找到剛剛解壓縮的 `ortho_gen-main` 資料夾，直接把它**拖拉**進終端機視窗裡。
   - 終端機此時會自動填入該資料夾的路徑，現在請按 `Enter` 鍵。

2. **建立執行環境**：
   - 複製這行文字並貼上到終端機：`python3 -m venv venv`
   - 按 `Enter` 鍵。(電腦可能會跑一下，請稍等)

3. **啟動環境**：
   - 複製這行文字並貼上：`source venv/bin/activate`
   - 按 `Enter` 鍵。
   - (成功的話，你會看到指令行的最前面出現 `(venv)` 的字樣)

4. **安裝必要工具**：
   - 複製這行文字並貼上：`pip install -r requirements.txt`
   - 按 `Enter` 鍵。
   - (這步會跑比較久，因為要從網路下載很多程式庫，請耐心等待直到跑完停止)

---

### 第四步：開始使用 (以後每次開啟)

每次要使用程式時，請執行以下步驟：

1. 開啟終端機。
2. 進入資料夾：輸入 `cd ` (加空白鍵) -> 拖拉資料夾 -> 按 `Enter`。
3. 啟動環境：輸入 `source venv/bin/activate` -> 按 `Enter`。
4. **執行程式**：
   - 輸入 **`python3 main.py`**
   - 按 `Enter` 鍵。
5. 當看到畫面上出現 `Starting web server...` 時，打開您的瀏覽器 (Safari 或 Chrome)。
6. 在網址列輸入：**`http://localhost:8080`** 並前往。

現在您應該可以在網頁上看到操作介面了！

### 如何關閉？

使用完畢後，回到終端機視窗，按下鍵盤 `Control` + `C` 即可停止程式。
